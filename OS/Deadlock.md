# Deadlock (교착 상태)

> [!NOTE] Deadlock
> 교착 상태는 **두 개 이상의 프로세스가 서로 필요로 하는 자원을 점유한 채, 다른 프로세스가 점유한 자원을 기다리느라 무한정 멈춰있는 상태**를 의미합니다. 마치 서로 다른 방향으로 가려는 두 차량이 좁은 길에서 서로 막혀 움직이지 못하는 상황과 같습니다.
