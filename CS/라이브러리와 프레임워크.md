
**라이브러리란?**

공통으로 사용될 수 있는 특정한 기능들을 모듈화한 것.
중복 코드 방지, 개발 속도 향상 [ex : #include<algorithm>의 sort 같은 것들]
즉, 필요할 때 내가 불러서 씀.

표준 라이브러리
- C++ STL, C# .NET Library
외부 라이브러리
- OpenGL, DirectX
- Boost.Asio
Unity에서 라이브러리
- TMPro, Newtonsoft Json, Firebase SDK


프레임워크란?

프로그램의 전체 구조와 흐름을 미리 정해 둔 개발의 뼈대.
