[[Algorithm]]
```c++
int main(void){
	int i,j,temp;
	int array[10] = {1,3,2,5,4,6,8,7,10,9};
	for(int i=0;i<10;i++) {
		for(int j=0;j<9-i;j++) {
			if(array[j] > array[j+1]) {
				temp = array[j];
				array[j] = array[j+1];
				array[j+1] = temp;
			}
		}
	}
}
```

* 옆에 있는 값과 비교해서 더 작은 값을 앞으로 보내면 어떨까

**시간복잡도**

O(N^2) => N ( N+1 ) / 2 => 대략 N * N
실제 수행시간이 선택정렬 보다 안 좋음. 정렬알고리즘 중에서 가장 느린 축에 속한다.
