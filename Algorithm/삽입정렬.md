[[Algorithm]]
```c++
int main(void){
	int i,j,temp;
	int array[10] = {1,10,3,4,5,6,7,3,4};
	for(int i=0;i<9;i++) {
		j=i;
			while(array[j] > array[j+1]) {// 하나씩 내려가면서 비교
			temp = array[j];
			array[j] = array[j+1];
			array[j+1]=temp;
			j--;
		}
	}
	return 0;
}
```

* 각 숫자를 적절한 위치에 삽입하면 어떨까
  항상 왼쪽에 있는 원소들이 이미 정렬되어 있다고 가정
	ex :     3 2 ->  _ 3 _ 2 -> 2 3 _ x
	
**시간복잡도**
정렬이 되어있을 수록 정렬 속도가 빠르다. 
O(N^2) => N ( N+1 ) / 2 => 대략 N * N
