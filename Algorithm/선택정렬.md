```c++
int main(void){
	int i,j,index,temp;
	int array[10] = {1,10,3,4,5,6,7,3,4};
	for(int i=0;i<10;i++) {
		min=9999;
		for(j=1;j<10;j++){
			if(min>array[j])
				min = array[j];
				index = j;
		}
		temp = array[i];
		array[i] = array[index];
		array[index] =temp;
	}
	return 0;
}
```

* 0번째 원소부터 순차적으로 N x N 번만큼 => (N (N+1)/2) 돌면서
  가장 작은 것을 선택해서 제일 앞으로 보내면 어떨까?

**시간복잡도**

O(N^2) => N ( N+1 ) / 2 => 대략 N * N
